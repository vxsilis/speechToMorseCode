“Speech to Morse Code” is a matlab app created by Vasileios Gavriilidis and Georgios Rossis and does exactly what it says!. It was developed under the “Speech Processing” course of Ionian University using the Google Cloud Speech to text API. The are 2 different language options and a lot of output customisation as frequency, character speed and transmission sound. All of the code as well as the installation guide is on GitHub.
